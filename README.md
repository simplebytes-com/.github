# Simple Bytes Organization Profile

This repository contains the public profile for the Simple Bytes organization on GitHub.

## What's in here?

- `profile/README.md` - The organization profile that appears on https://github.com/simplebytes-com

## How to update

Edit `profile/README.md` and commit changes to update the organization profile.

For more information, see [GitHub's documentation on customizing organization profiles](https://docs.github.com/en/organizations/collaborating-with-groups-in-organizations/customizing-your-organizations-profile).
